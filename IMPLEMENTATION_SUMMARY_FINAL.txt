================================================================================
                        ORDER CHANNELS TAB
================================================================================

Location: Admin Panel → Order Channels Tab (Send icon, purple)

THREE OPTIONS (SELECT ONE):

1. WhatsApp Channel
   ├─ Purpose: Send order details via WhatsApp message
   ├─ Config: API Key + Business Phone Number
   ├─ Template: {ORDER_ID}, {TOTAL}, {STATUS}
   └─ Status: Saves to Firebase when selected

2. Telegram Channel
   ├─ Purpose: Admin receives order notifications
   ├─ Config: Bot Token + Chat ID
   ├─ Template: {ORDER_ID}, {CUSTOMER_NAME}, {TOTAL}
   └─ Status: Saves to Firebase when selected

3. Payment Gateway (Default)
   ├─ Purpose: Traditional checkout flow
   ├─ Config: None (built-in)
   ├─ Template: Standard email
   └─ Status: Default if no other selected

Radio Button Selection (ONLY ONE):
⦿ WhatsApp      (ACTIVE - show config)
⚫ Telegram      (INACTIVE - hidden)
⚫ Payment       (INACTIVE - hidden)

================================================================================
                       HOW TO USE
================================================================================

ADMIN USER: Configure Order Notifications
1. Go to Admin Panel
2. Click "Order Channels" tab (Send icon)
3. Select ONE:
   ⦿ WhatsApp (to send order messages)
   ⚫ Telegram (to receive notifications)
   ⚫ Payment (standard checkout)
4. Enter API Key / Credentials
5. Customize message template
6. Click "Save Order Channel Configuration"
7. Configuration saved to Firebase
8. Orders will use this channel
